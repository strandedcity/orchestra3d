Orchestra3d.io
 

Orchestra3d.io is a generative NURBS modeler for the web. Its familiar visual programming interface is designed to make it accessible for designers. It runs incredibly fast, all in your web browser, and without connecting to a server. Your web browser is employed as a fully functional platform for CAD, making intuitive NURBS-based parametric modeling accessible to everyone everywhere, free of charge.

The following preview demonstrates the ease with which large fields of geometry can be controlled by a single "attractor point". The parametric definition is built from scratch, then put to use right away.

Features

  • Intuitive NURBS Geometry
  • Familiar Visual Programming
  • STL and SVG Export
  • GHX Import
  • Share Designs with Anyone
  • Direct Upload to Shapeways
  • Runs Anywhere in your Web Browser
  • No Downloads Required
